# Product Knowledge — Discovery Router
> Read this when: building features, writing copy, answering questions about the product, or designing UX flows.

## What It Is

The Discovery Router is an AI-powered project scoping tool that turns vague ideas into structured SPEC documents. It detects the right discovery approach and guides users through tailored questions.

## The Four Routes

| Route | When | User Profile | Approach |
|-------|------|-------------|----------|
| **A — Standard** | Clear requirements, needs documentation | PM or tech lead | Structured Q&A → SPEC |
| **B — Exploratory** | Vision needs clarifying | Founders, innovators | Prototype-first, open-ended |
| **C — Stakeholder** | Multiple decision-makers | Enterprise, agencies | Alignment-focused, priority mapping |
| **D — Integration** | Connecting existing systems | CTOs, dev teams | API/system mapping, dependency analysis |

## Two Modes
1. **Guided Mode**: 5-question flow → automatic route detection → SPEC preview
2. **AI Chat Mode**: Free-form conversation with Claude (filesystem agent pattern)

## Freemium Flow
1. User starts free discovery (guided or chat)
2. Route detected automatically from answers
3. **Free tier**: SPEC preview (project summary + 2-3 key features)
4. **Paid tier** (R2,500): Full SPEC.json with complete requirements, risk analysis, timeline
5. Email capture at preview stage

## Pricing
- **Free**: Discovery preview (lead gen)
- **Discovery Toolkit**: R2,500/session (full SPEC)
- **Consulting add-on**: Custom pricing
- **Brand Kit exports** (via finya.one): R250/export

## Key Differentiators
- Catches scope creep BEFORE it happens (red flags, signals)
- Filesystem agent pattern — not generic chatbot
- Route-specific playbooks refined from real project data
- SA-market focused (ZAR, PayFast, local pain points)
